## **워드클라우드 생성 프로그램 설명**

이 프로그램은 네이버 검색 페이지에서 특정 게임의 줄거리를 검색한 결과를 크롤링하고, 해당 결과에서 가장 자주 사용되는 단어들을 추출하여 워드클라우드로 시각화하는 프로그램입니다.

### **사전 설정**

먼저, 필요한 라이브러리를 설치하고, 크롬 드라이버를 설정해주어야 합니다. 아래 명령어를 실행하여 관련 패키지들을 설치합니다:

### **코드 실행**

1. 먼저, **`loadDriver()`** 함수를 사용하여 웹 드라이버를 설정합니다. 이 함수는 크롬 드라이버를 headless 모드로 실행시켜줍니다.
2. **`loadDescription()`** 함수는 네이버 검색 결과 페이지의 특정 URL을 받아와 해당 페이지의 본문 내용을 추출합니다.
3. 주어진 검색어로 네이버 검색 페이지에서 크롤링한 본문 내용을 **`lines`** 변수에 저장합니다.
4. 그 후, **`matplotlib`** 라이브러리를 사용하여 워드클라우드를 생성합니다. 워드클라우드에는 자주 사용되는 단어들이 시각화되어 표현됩니다.
5. 마지막으로, **`konlpy`** 라이브러리를 사용하여 자주 사용되는 명사를 추출하고, 이를 빈도 순으로 정렬하여 상위 20개를 출력합니다.

### **주의사항**

- 이 프로그램은 Colab 환경에서 실행되도록 작성되었습니다. 로컬 환경에서 실행할 경우 일부 경로와 패키지 설치 명령어가 변경되어야 할 수 있습니다.
- 워드클라우드를 생성하는데 사용된 **`WordCloud`** 클래스는 한글 폰트를 지원하지 않기 때문에, 별도의 한글 폰트 설정을 해주어야 합니다. 해당 코드에서는 나눔바른고딕 폰트를 사용하였습니다. 필요에 따라 다른 한글 폰트로 변경할 수 있습니다.
- 이 코드는 네이버 검색 결과 페이지의 HTML 구조를 기반으로 작성되었습니다. 네이버의 페이지 구조가 변경되면 프로그램이 작동하지 않을 수 있으며, 이 경우에는 코드를 업데이트해야 합니다.
- 웹 크롤링은 해당 웹 사이트의 이용 약관을 준수해야 하며, 너무 빈번하게 요청하거나 과도한 트래픽을 유발하지 않도록 주의해야 합니다.
